Interactive Tutorial

Interactive lessons for CS 348 — Midterm 2

A SQLBolt-style tutorial built from the lecture handouts: Module 3 (Basic SQL), Module 4 (Relational Algebra) and Module 5 (Advanced SQL). Every lesson uses the lecture schemas — the bibliography database and the ACCOUNT/BANK instance — on an extended data set, with exercises pitched at midterm level. Queries run live in your browser as you type: SQL on SQLite (via sql.js), and relational algebra on a purpose-built σ π ρ × ∪ − interpreter.

Work through the lessons in order, or jump to a topic. Your progress is saved in the browser.
